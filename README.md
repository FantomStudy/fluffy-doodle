Проект состоит из фронтенда, бэкенда и мобильного приложения.

## Запуск фронтенда и бэкенда

Из корня проекта выполните:

```bash
docker-compose up --build
```

После запуска фронтенд будет доступен по адресу:

```text
http://localhost:5173
```

## Запуск мобильного приложения в браузере

1. Перейдите в папку `web/mobile`
2. Установите зависимости:

```bash
npm i
```

Если `npm` не установлен, сначала установите Node.js и npm.

3. Запустите Expo в web-режиме:

```bash
npx expo start --web
```

Мобильное приложение откроется в браузере, поэтому не нужно собирать отдельный билд для телефона.

Чтобы авторизоваться как родитель, нужно взять в профиле ребенка его уникальный код, при регистрации поставить галочку, что вы родитель и ввести этот код.
